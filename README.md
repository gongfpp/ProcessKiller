# ProcessKiller
Use Qt to established a ProcessKiller with Gui


点击查找进程将目前所有进程显示，点击“杀杀杀杀”按钮弹出模态对话框，输入欲杀进程的PID，即可干烂（此图中PID为自己，即自杀）
查找进程按钮杀鸡用牛刀用了多线程。核心功能为调用cmd的tasklist和taskkill。
实现总时间约为2小时，一小时写多线程，一小时解决中文乱码问题。
